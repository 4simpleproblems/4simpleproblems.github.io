## What is 4SP?

4SP (4 Simple Problems) is a browser-based game site built for speed and simplicity.

Version 5 is a full rewrite.
Cleaner structure.
Better UI.
Easier deployment.

No installs.
No downloads.
Just the web.

## Features

1. Fork the repository
2. Upload files to any static host
3. Set the root directory
4. Deploy

## Local Development

```bash
python -m http.server
```

or

```bash
npx serve
```

Open `http://localhost:8000`.
