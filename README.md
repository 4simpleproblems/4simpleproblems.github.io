## What is 4SP?

4SP (4 Simple Problems) is a browser-based game site focused on speed, simplicity, and clean UI.
Version 5 is a full rewrite with better structure, theming, and deployment support.

No installs. No downloads. Just the web.

## Features

<div align="center">

|    Fast UI    | Theme Support |  Modular Pages |
| :-----------: | :-----------: | :------------: |
|  Low Latency  |  Light / Dark | Easy Expansion |
| Simple Layout |   Responsive  |  Clean Routing |

</div>

## Tech

* HTML
* CSS
* JavaScript
* Static-first architecture
* Deployable anywhere that serves web files

## Deployment

### Easy Deployment

You can deploy this project instantly using the buttons below.

#### Vercel

```md
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/import?framework=other&s=https%3A%2F%2Fgithub.com%2F4simpleproblems-v5%2F4simpleproblems-v5.github.io)
```

#### Netlify

```md
[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/4simpleproblems-v5/4simpleproblems-v5.github.io)
```

#### Koyeb

```html
<a target="_blank" href="https://app.koyeb.com/deploy?type=git&repository=github.com/4simpleproblems-v5/4simpleproblems-v5.github.io">
  <img alt="Deploy to Koyeb" src="https://www.koyeb.com/static/images/deploy/button.svg">
</a>
```

### Manual Deployment

1. Fork the repository.
2. Upload files to any static host.
3. Set the root directory to the project folder.
4. Deploy.

## Development

Local testing:

```bash
python -m http.server
```

or

```bash
npx serve
```

Then open `http://localhost:8000`.
